# Autism Health Tracker

## Project Features
- **User Profiles**: Create and manage user profiles for individual tracking.
- **Symptom Tracking**: Log symptoms over time and visualize trends.
- **Medication Management**: Keep track of medications, dosages, and schedules.
- **Reporting**: Generate reports for healthcare providers.

## Accessibility
- **Keyboard Navigation**: All functionalities are accessible via keyboard shortcuts.
- **Screen Reader Compatibility**: The application is designed to be compatible with screen readers.

## How to Use the Templates
1. **Clone the Repository**: Use `git clone <repository-url>` to clone the project locally.
2. **Install Dependencies**: Run `npm install` to install required packages.
3. **Run the Application**: Use `npm start` to launch the application.
4. **Use the Templates**: Follow the in-app guides to use the templates effectively.
